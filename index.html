GAN Generator Loss — Gradient Behavior

A generator learns by receiving gradient feedback. But the gradient only exists if the loss curve has a slope.
Drag the slider to see what happens to that slope depending on which loss you use.

Minimax Loss L = log(1 − D(G(z))) original GAN

When the discriminator is confident the image is fake — the left side of the chart — the curve is nearly flat. A flat curve means the gradient is close to zero, so the generator receives almost no learning signal and training stalls.
